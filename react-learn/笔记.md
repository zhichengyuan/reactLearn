# 属性默认值 和 类型检查

## 属性默认值

通过一个静态属性```defaultProps```告知react属性默认值

## 属性类型检查···

使用库：```prop-types```

对组件使用静态属性```propTypes```告知react如何检查属性

```js

PropTypes.any: //任意类型
PropTypes.array: //数组类型
PropTypes.bool: //布尔类型
PropTypes.func: //函数类型
PropTypes.number: //数字类型
PropTypes.object: //对象类型
PropTypes.string: //字符串类型
PropTypes.symbol: //符号类型

```
